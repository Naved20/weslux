## 1. Account Creation & Onboarding

### Step 1: Sign Up
- **Access**: Visit platform → Click "Sign Up"
- **Process**:
  1. Enter basic details: Name, Email, Password
  2. Select user type: "Mentee"
  3. Verify email via OTP
  4. Set up password

### Step 2: Initial Login
- **Access**: Email verification → Login with credentials
- **Process**:
  1. Enter email and password
  2. Complete 2-factor authentication (if enabled)
  3. Redirect to mentee dashboard

### Step 3: Profile Completion Prompt
- **System Action**: Automatic redirect to profile completion page
- **Requirement**: Must complete profile before accessing full features
- **Grace Period**: Limited access until profile completion

## 2. Profile Setup & Completion

### Mandatory Profile Fields
1. **Personal Information**
   - Full Name (pre-filled from signup)
   - Email (pre-filled from signup)
   - Date of Birth

6. **Agreements**
   - Terms & Conditions (mandatory)

7. **Profile Picture** (mandatory)

## 4. Mentorship Types Explained

### Long-term Mentorship

#### Definition
A structured, ongoing relationship focused on comprehensive development over an extended period (typically 3-12 months).

#### Characteristics
- **Duration**: 3-12 months
- **Frequency**: Regular weekly/bi-weekly meetings
- **Focus**: Holistic development, career growth, skill building
- **Structure**: Formal agreement, defined goals, progress tracking

#### Ideal For
- Career transition planning
- Leadership development
- Skill mastery
- Professional certification preparation
- Long-term career strategy

### Short-term Mentorship

#### Definition
A focused, time-bound engagement addressing specific challenges or goals (typically 1-4 weeks).

#### Characteristics
- **Duration**: 1-4 weeks
- **Frequency**: 1-3 sessions
- **Focus**: Specific problem-solving, quick wins
- **Structure**: Task-oriented, immediate results

#### Ideal For
- Resume review
- Interview preparation
- Specific skill coaching
- Project guidance
- Quick career advice
- Networking introduction
### Comparison Table

| Aspect | Long-term Mentorship | Short-term Mentorship |
|--------|---------------------|----------------------|
| **Duration** | 3-12 months | 1-4 weeks |
| **Commitment** | High | Low to Medium |
| **Focus** | Holistic development | Specific problem-solving |
| **Structure** | Formal, structured | Flexible, task-oriented |
| **Relationship** | Deep, ongoing | Transactional, focused |
| **Goals** | Multiple, long-term | Single, immediate |
| **Cost** | Higher investment | Lower investment |
| **Best For** | Career transitions, skill mastery | Quick wins, specific guidance |

## 5. Mentorship Request Process

### Step 1: Mentor Selection
- Browse mentor profiles
- Review mentor details, ratings, availability
- Select preferred mentorship type (Long-term/Short-term)

### Step 2: Request Creation
- **Access**: Mentor profile → "Request Mentorship"
- **Form Fields**:
  1. **Mentorship Type**: Long-term/Short-term
  2. **Purpose**: Clear objective for mentorship
  3. **Why This Mentor**: Specific reasons for selection
  4. **Expected Outcomes**: What you hope to achieve
  5. **Preferred Schedule**: Availability preferences
  6. **Duration**: Estimated timeline
  7. **Additional Notes**: Special requirements

### Step 3: Request Submission
- **System Actions**:
  1. Request sent to mentor
  2. Notification to mentee (confirmation)
  3. Request appears in "Pending Requests"
  4. Supervisor notification (if institution-based)
